Реализовать систему классов для работы светофора.

Светофоры бывают: 
- с 3-мя (красный, желтый, зелёный)
- 2-мя (красный, зелёный) цветами. 

К функциям светофора относятся: 
- зажигание определённого цвета вручную; 
- зажигание поочередно цветов по таймеру; 
- отключение всех цветов.

Для реализации желательно использовать Windows Forms на C#. 
Допустима реализация с использованием языка Java (C# и Java похожи). 
Рисовать сам светофор не требуется, можно просто выводить текущий цвет текстом: в консоль, в GUI.
